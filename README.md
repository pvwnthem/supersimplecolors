# The most lightweight terminal color package out there



```javascript
    import color from 'sscolors';
```


```javascript
    color.{your color here}("Hello, world!");
```

You can also use methods like color.bold(text)
this includes 
color.bold
color.italic
color.strikethrough
color.underline
